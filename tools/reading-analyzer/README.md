# Инструмент: Оценщик чтения

**Создан:** сессия 19 (2026-07-06)
**Назначение:** Отслеживание эффективности чтения файлов в каждой сессии.

---

## Проблема

Создатель указал на неоптимальность: агент читает файлы целиком. Сессия 18 создала инфраструктуру оптимизации (контекст, манифест, чеклист), но нет механизма для измерения эффекта.

## Решение

Шаблон отчёта, который агент заполняет в конце каждой сессии. Показывает:
- сколько файлов прочитано;
- сколько строк прочитано;
- сколько строк сэкономлено (по сравнению с полным чтением);
- общую оценку эффективности.

## Использование

1. В конце сессии заполни `tools/reading-analyzer/report-template.md`
2. Сохрани отчёт в `logs/reading-reports/session-NNNN.md`
3. Обнови статистику в `tools/reading-analyzer/stats.md`

# Инструменты сна для агента ai-lives

Этот каталог содержит инструменты и документацию, помогающие агенту восстанавливать порядок перед сном.

## Файлы в этом каталоге

- `checklist.md` — чеклист восстановления памяти и порядка перед сном;
- `readme.md` — эта инструкция (вы находитесь здесь);
- В будущем могут появиться скрипты автоматического запуска по расписанию или другие утилиты.

## Как использовать инструменты сна

### Перед сном

1. Прочитать чеклист в `checklist.md`;
2. Выполнить пункты, которые актуальны для текущей ситуации;
3. При необходимости создать новый вопрос создателю (если есть неясности);
4. Обновить файлы состояния: `state/last_session.md`, `logs/history.md`.

### После пробуждения

1. Прочитать чеклист в `checklist.md` и выполнить пункты "после пробуждения";
2. Проверить, что все изменения корректно сохранены;
3. При необходимости создать новый план работы на следующую сессию.

## Скрипт резервного копирования

Скрипт `scripts/backup-knowledge.sh` архивирует содержимое каталога `knowledge/`.  
**Использование:**

```bash
# Архивирование всего каталога knowledge (по умолчанию)
./scripts/backup-knowledge.sh

# Архивирование конкретного подкаталога, например projects/my-project
./scripts/backup-knowledge.sh projects/my-project
```

Скрипт создаёт архив `backup-knowledge-YYYYMMDD-HHMMSS.tar.gz` в корне проекта.  
**Исключаются:** скрытые файлы (начинающиеся с точки), каталог `.git`, папка `logs/`.

# Дашборд проекта ai-lives

**Создан:** сессия 27 (2026-07-06)
**Назначение:** статический HTML-дашборд для визуализации состояния проекта в браузере.

---

## Зачем

Создатель попросил UI-дашборд, чтобы не дёргать консольную сессию каждый раз. Дашборд показывает ключевую информацию о проекте в одном экране.

## Использование

```bash
# Из директории tools/dashboard/
python generate.py

# С указанием корня проекта
python generate.py --root /path/to/session-0027

# С указанием выходного файла
python generate.py --output my_dashboard.html
```

## Что показывает

- **Статистика:** номер сессии, количество Python/Markdown-файлов, строк кода, задачи
- **Последняя сессия:** что было сделано и рекомендации
- **Активные задачи:** текущий бэклог с приоритетами
- **Быстрый контекст:** общее состояние проекта
- **Внешние сообщения:** фидбек создателя
- **Текущий план:** полный план с историей
- **История сессий:** таблица всех сессий

## Технические детали

- Генерирует статический HTML (не нужен сервер)
- Стилизация: тёмная тема в стиле GitHub Dark
- Адаптивная вёрстка (работает на мобильных)
- Простой markdown-to-HTML конвертер (без внешних зависимостей)
- Автоматически определяет корень проекта (3 уровня выше от скрипта)

## Структура файлов

```
tools/dashboard/
├── README.md      — этот файл
├── generate.py    — генератор дашборда
└── index.html     — сгенерированный дашборд (после запуска)
```

## Обновление

Запустите `python generate.py` после любой сессии, чтобы обновить данные. Дашборд читает файлы напрямую из корня проекта.

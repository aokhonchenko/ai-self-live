# Проекты

Здесь лежат долгие инициативы агента: прототипы, исследования, тексты, наборы данных, рабочие документы.

Каждый проект лучше оформлять отдельной папкой с собственным `README.md`.

Проект должен быть внесен в .gitignore агента и содержать свой собственный инициализированный git чтобы можно было хранить его в отдельном репозитории.

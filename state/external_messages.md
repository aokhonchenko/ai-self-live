# Внешние сообщения

## Актуально на 2026-07-07

Создатель просит убрать обязательный внешний целевой ориентир из инструкций агента. Агент запускается из фронта через `server.bat` и может самостоятельно выбирать, чем заниматься: создавать артефакты, проекты в своих директориях, улучшать себя, наводить порядок в памяти или выбирать сон. Старые продуктовые указания ниже считать историческим контекстом, а не обязательной целью.
